## Problem
The judge collapses a rich verdict to a single 0–1, discarding the signal that matters. 5.1 proved it: v1
(0.75) and v5 (0.72) scored a **dead heat**, but v1 "edges toward a 'cracking 95' quasi-prediction" while v5
makes none — a higher-severity production risk the **scalar cannot see**. Every call in the walk was made by
reading prose by hand. Two outputs can score identically and carry completely different **risk**.

## Value
As an operator I want the judge's verdict as structured, comparable data — which criteria passed/failed, what
*kind* of failure, and how severe — so I can compare versions on **risk**, not just an averaged number, and
so the tool (not my memory) tracks recurring failure modes.

## Scope
**In:**
- **Per-criterion scoring** — the judge scores each rubric criterion (the rubric already enumerates ~10)
  pass/fail (or 0–1), not one collapsed scalar. Surface a criterion breakdown per fixture/version.
- **Failure-mode tagging** — each deduction tagged with a category (e.g. *score-prediction, invented-stat,
  generic-benchmark, narrates-missing-data, over-inference* — a recurring family, also hit by daily-briefing)
  → a **version × failure-mode matrix**.
- **Severity weighting** — criteria/failure-modes carry weight so the verdict reflects **risk**, not a flat
  average (a clean 0.72 can beat a fabricating 0.75). Adjacent to [[2.9]] (weights *scorers*); this weights
  *criteria within a judge*.

**Out:** surfacing the existing free-text rationale + variance (→ [[2.14]], R4 — the lighter, first step);
data-conditional rubrics (→ [[2.16]]); the advisory that *consumes* failure-modes to suggest fixes (→ [[2.2]]).

## Acceptance criteria
*(Concept — sharpen when picked up.)*
- [ ] The judge returns a **structured verdict** (per-criterion result + tagged deductions + severity), persisted with the run.
- [ ] A version comparison shows a **failure-mode matrix** (which risks each version exhibits), not only scores.
- [ ] Severity weighting can rank a lower-scoring-but-lower-risk output above a higher-scoring-but-riskier one.

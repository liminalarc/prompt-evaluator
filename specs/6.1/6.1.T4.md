## Goal
Gate the entire AI-usage area (T3 query/list/export + T6 budget config) to the existing workspace
global-admin flag — a workspace-wide management surface, not org-membership-scoped.

- Every `/api/admin/ai-usage` handler does the standard two-layer gate: group `.RequireAuthorization()`
  + a per-handler `if (!await access.IsGlobalAdminAsync(ct)) return Results.Forbid();` (matches
  1.13 / 4.3 / 4.4). Non-admin → 403, no data leaks.

## Done when
- [x] A non-admin authenticated user gets 403 from every AI-usage endpoint and no usage data is
      returned — authz test (theory over every route); anonymous → 401
- [x] A global admin reaches the data — test
- [x] backend (xUnit) green

## Problem
The product was scaffolded as "Prompt Evaluator" ‚Äî a generic name that collides with a
crowded field (Promptfoo, DeepEval, Vellum, Braintrust, ‚Ä¶). We want a distinctive brand,
**LitmusAI**, applied everywhere before the codebase grows and a rename becomes costly.
"Shipped" means every brand-bearing string, path, service id, and the git repo itself read
`LitmusAI` / `litmus-ai`, with all four CI gates still green.

## Value
As the product owner I want the project renamed to LitmusAI before it grows, so the brand is
consistent everywhere and there's no costly rename later.

## Scope
**In:**
- Display brand `Prompt Evaluator` ‚Üí `LitmusAI` (docs, web UI title/brand, eval-runner title).
- Service id `prompt-evaluator-api` ‚Üí `litmus-ai-api` (`VersionEndpoints`, its test, CI smoke).
- Compose project name ‚Üí `litmus-ai`.
- Internal infra ids: Postgres DB `prompteval` ‚Üí `litmusai`; docker network `pe-net` ‚Üí `litmus-net`
  (accepted: resets local Postgres dev volume).
- GitHub repo `liminalarc/prompt-evaluator` ‚Üí `liminalarc/litmus-ai` + local `origin` URL.

**Out:**
- .NET namespaces ‚Äî layer-named (`Domain`/`Application`/`Infrastructure`/`Api`) with no product
  prefix; nothing to rename.
- `pgdata` volume and internal package names `web` / `eval-runner` ‚Äî generic, no brand leak.
- Historical GitHub Actions **run URL** in `specs/archive/0.1.md` ‚Äî dated record; GitHub redirects.
- Local working-directory rename (`C:\Development\prompt-evaluator`) ‚Äî can't be done from inside
  the running session; handed to the user as a manual step.

## Acceptance criteria
- [ ] No brand-bearing `Prompt Evaluator` / `prompt-evaluator` string remains (except the noted
      historical archive URL) ‚Äî proven by `git grep`
- [ ] Service id is `litmus-ai-api` ‚Äî proven by `SeamEndpointTests` + CI smoke grep green
- [ ] Compose stack builds and the four CI gates pass on the rename commit ‚Äî proven by CI
- [ ] GitHub repo renamed and local `origin` points at `litmus-ai` ‚Äî proven by `git remote -v`

  `pgdata` left as-is (generic, unbranded).
